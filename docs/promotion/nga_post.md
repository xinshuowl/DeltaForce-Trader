各位大佬好，先免责：**本工具仅供学习交流，使用风险自负**，建议小号体验。

最近被手动上架交易行折磨疯了 — 仓库满了之后一件一件点过去要 30 多分钟，眼睛都花。
于是花了点时间写了个自动化工具，开源出来给同样懒得手动的朋友用。

## 它能做什么

- **一键挂上交易行**：按 Ctrl+1，自动遍历你勾选的箱子，把所有非绑定道具拉满数量挂上交易行
- **挂机循环**：上满 15 槽后等几分钟，自动下架已成交的，整理仓库，重新挂下一轮，全程不用人
- **精确筛选**：能按品质（紫色/金色等）+ 分类（枪械/弹药/装备等）+ 仓库箱子三轴筛
- **不重复点同一个道具**：枪占 8 格也只点一次（很多老脚本会反复点）
- **收入自动统计**：跑完会列出每件道具的预期收入和总额

## 运行环境

- Windows 10/11
- 分辨率 **2560×1440 + 150% 缩放**（其他分辨率要校准坐标）
- 必须**管理员权限运行**（按键监听需要）
- 游戏开**无边框窗口**

## 设计上的几个亮点

- **ML 模型识别绑定/非绑定**：不靠固定颜色阈值，新版本游戏改 UI 也能用
- **连通域分析**：一把枪占多格也能识别成 1 个物品
- **三层去重**（位置+名字+图像指纹）：跨页跳过已经看过的物品，不会鬼打墙
- **坐标可视化校准**：30+ 个坐标点全在 GUI 里可视化调，不用改代码

## 备用回帖话术

### 楼下问"会不会封号"

回：脚本类工具理论上都有封号风险。我自己跑了 N 小时小号没事，但不代表你的号也安全。建议：

1. 先在小号上试
2. 不要长时间挂机（建议单次循环 < 2 小时）
3. 不要在凌晨 / 维护前后用（官方风控最严的时段）

### 楼下问"代码安全吗"

回：完全开源，源码全在 GitHub 上随便看。打包好的 exe 也是 GitHub Actions 自动从源码构建的，可以追溯每一行：
https://github.com/xinshuowl/DeltaForce-Trader/actions

### 楼下问"用什么写的"

回：Python + PyQt5（界面）+ OpenCV（图像）+ scikit-learn（ML）+ pyautogui（鼠标键盘）。打包用 PyInstaller。

### 楼下骂"这不就是脚本吗"

回：是的，所以名字叫"自动化工具"不叫"挂"。开源免费，源码可查，不动游戏内存，纯模拟鼠标键盘。要不要用看你自己。
